# Contributing

Keep changes small, practical, and source-grounded.

- Do not commit generated catalog references.
- Keep recommendations educational and non-prescriptive.
- Prefer improving search, ranking, citations, safety framing, and extraction quality.
- Run `node scripts/build_catalog_reference.js` and `node scripts/run_smoke_tests.js` before submitting changes.
- Avoid adding dependencies unless they materially improve parsing, ranking, or verification.
